# Инструкция по деплою на GitHub Pages

## Шаг 1: Создание репозитория на GitHub

1. Перейдите на [GitHub.com](https://github.com) и войдите в свой аккаунт
2. Нажмите кнопку **"New"** (или **"+"** → **"New repository"**)
3. Заполните форму:
   - **Repository name**: `ai-key` (или любое другое имя)
   - **Description**: описание проекта (опционально)
   - Выберите **Public** (для бесплатного GitHub Pages)
   - **НЕ** добавляйте README, .gitignore или лицензию (они уже есть в проекте)
4. Нажмите **"Create repository"**

## Шаг 2: Инициализация Git и загрузка кода

Выполните следующие команды в терминале (из директории проекта):

```bash
cd /Users/dr/Downloads/ai-key

# Инициализация git репозитория
git init

# Добавление всех файлов
git add .

# Первый коммит
git commit -m "Initial commit: AI Key landing page"

# Добавление remote репозитория (замените YOUR_USERNAME на ваш GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/ai-key.git

# Переименование ветки в main (если нужно)
git branch -M main

# Загрузка кода на GitHub
git push -u origin main
```

## Шаг 3: Настройка GitHub Pages

1. Перейдите в ваш репозиторий на GitHub
2. Откройте **Settings** (вкладка вверху)
3. В левом меню найдите **Pages**
4. В разделе **Source** выберите:
   - **Source**: `GitHub Actions`
5. Сохраните изменения

## Шаг 4: Автоматический деплой

После настройки GitHub Actions будет автоматически деплоить ваш сайт при каждом push в ветку `main`.

**Важно**: При первом деплое может потребоваться несколько минут.

## Шаг 5: Доступ к сайту

После успешного деплоя ваш сайт будет доступен по адресу:
- `https://YOUR_USERNAME.github.io/ai-key/`

(Замените `YOUR_USERNAME` на ваш GitHub username и `ai-key` на имя вашего репозитория)

## Настройка кастомного домена (опционально)

Если вы хотите использовать свой домен:

1. В `vite.config.ts` измените `base: '/ai-key/'` на `base: '/'`
2. В настройках GitHub Pages добавьте ваш кастомный домен
3. Настройте DNS записи у вашего регистратора домена

## Обновление сайта

Для обновления сайта просто выполните:

```bash
git add .
git commit -m "Описание изменений"
git push
```

GitHub Actions автоматически соберет и задеплоит новую версию.

## Проверка статуса деплоя

1. Перейдите в ваш репозиторий на GitHub
2. Откройте вкладку **Actions**
3. Там вы увидите статус всех деплоев

## Возможные проблемы

### Сайт не открывается
- Убедитесь, что деплой завершился успешно (вкладка Actions)
- Проверьте, что в `vite.config.ts` указан правильный `base` путь
- Подождите несколько минут после первого деплоя

### Неправильные пути к ресурсам
- Убедитесь, что `base` в `vite.config.ts` соответствует имени репозитория
- Если репозиторий называется `ai-key`, base должен быть `/ai-key/`

### Ошибки сборки
- Проверьте логи в разделе Actions
- Убедитесь, что все зависимости указаны в `package.json`
- Проверьте, что нет синтаксических ошибок в коде
